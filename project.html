The History of Baseball (SQL, python)
Fall 2019

I found two databases with common features. The first dataset contains information about Major League Baseball (MLB) in the United States. The second dataset contains information about Nippion Professional Baseball (NPB) in Japan. Each file in each dataset had at least 5,000 oberservations and at least 5 features.

The datasets were modeled using BigQuery to at least 2nd normal form. Inconsistent naming conventions in the datasets were fixed using Apache Beam and Dataflow. These fixes to the datasets were also demonstrated in Apache airflow. Graphs were made in Google Data Studio to visualize the data as well as differences between baseball in the United States compared to baseball in Japan.

The History of Home Runs (R)
Fall 2019

Controlling for salary, does the linear trend predicting number of home runs hit every year change through time? Additionally, does batting hand significantly moderate the effect of year on home runs and/or does league moderate the effect of year on home runs?

To answer this question, I used segmented regression (piecewise regression) with interactions. I used data from a baseball database compiled by author and journalist, Sean Lahman. This database contains baseball statistics from 1871 - 2018.

Good Bois Website (HTML, CSS, PHP, JavaScript)
Spring 2019

Every animal deserves a home and family that cares for it. "Good Bois Social" is a way to show off your pet to others and view other pet owner's animals. Posting and viewing Good Bois Social is only accessible to members who make accounts on the website. "Good Bois Adopt" allows you to enter in your zip code and find animals at animal shelters in your area.

This webiste features dynamic HTML, a MySQL database, PHP interface to database, sessions, cookies, an ajax application, and safe web programming practices.

Good Bois iOS App (Swift)
Spring 2019

This app features core graphics, core data storage, core location, gestures, and networking functionality. Similar to the website, the user is able to find dogs in animal shelters based on location and other preferences. The user is able to swipe right and left on dogs they like or dislike and save the pets to view later.

The API used was through PetFinder. The API used OAuth for authorization and authentication.

Platform Game (Python)
Fall 2018

This project uses unit testing, acceptance testing, refactoring, and other elements of Xtreme Programming.

This project is a 2D side-scrolling game. The game utilizes file manipulation by saving a text file with player information such as the user’s lives remaining, current level, and score. The game updates this text file as the user progresses in the game. When the user continues the game, the program loads the text file. The user is able to shoot enemies and interact with objects while exploring their surrounding.
